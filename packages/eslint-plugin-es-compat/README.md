# eslint-plugin-es-compat

> ESLint plugin for checking JavaScript code compatibility with target runtime environments

```bash
npm install --save-dev eslint-plugin-es-compat
```

```json
// .eslintrc.json
{
  "plugins": ["es-compat"],
  "rules": {
    "es-compat/compat": "error"
  }
}
```

```
// .browserslistrc
Chrome >= 64
Firefox >= 58
```
